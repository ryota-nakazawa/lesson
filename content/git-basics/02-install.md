# はじめの準備

最初に必要なのは、Gitを使える状態にすることです。すでに入っている環境も多いので、まずは確認します。

- ターミナルを開く
- `git --version` を実行する
- バージョンが表示されれば準備済み

表示されない場合は、OSに合わせてGitをインストールします。

## 最初のリポジトリ

Gitで履歴を管理する対象のフォルダを **リポジトリ** と呼びます。

```txt
project/
  README.md
  index.html
```

このフォルダで `git init` を実行すると、そのフォルダがGit管理の対象になります。

> チーム運用では、最初にREADMEを用意して「何のためのリポジトリか」を明記しておくと迷子になりにくいです。
