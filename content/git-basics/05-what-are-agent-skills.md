# Agent skillsって何？

Agent skillsは、AIエージェントに特定の作業手順や判断基準を覚えさせるための仕組みです。

毎回同じ説明をしなくても、スキルとしてまとめておくことで、AIがその作業に合った進め方をしやすくなります。

> まとめ: Agent skillsは、AIに特定作業の進め方を渡すための手順書です。

## なぜスキルが必要か

AIは汎用的に会話できますが、プロジェクト固有の作法までは最初から知りません。

たとえば、次のようなことはスキル化すると便利です。

- 学習コンテンツの作り方
- チェックテストの作り方
- ドキュメントのレビュー観点
- 特定リポジトリでの検証手順

## 何を書くのか

スキルには、主に次のような情報を書きます。

- いつ使うか
- どういう手順で進めるか
- どのファイルやコマンドを見るか
- 完了前に何を確認するか

このプロジェクトでも、学習コンテンツ管理やチェックテスト作成の流れをスキルとして管理しています。

## FDE人材が押さえること

生成AIを業務で使うときは、毎回プロンプトで全部説明するよりも、再利用できる手順をスキル化した方が安定します。

```txt
毎回説明する
  ↓
よく使う手順をスキル化する
  ↓
同じ品質で繰り返しやすくなる
```

> Agent skillsは、AIとの共同作業を属人的な会話から再利用できる手順へ近づける仕組みです。
